# Tourist Guide Web Portal 🌍✈️

An interactive and feature-rich web application designed to help travelers discover destinations, manage trips, and share their experiences. This project consists of a full user portal and a dedicated administration dashboard.

## 🌟 Key Features

* **Interactive Mapping:** Features an interactive world map powered by Leaflet.js allowing users to explore locations visually.
* **Dual Interface System:** 
  * **User Portal:** Allows travelers to browse destinations, read reviews, and save locations to a personal wishlist.
  * **Admin Dashboard:** Provides administrators the ability to dynamically add new travel destinations, complete with mandatory fields for accommodation, transportation, and food images.
* **Trip Management:** Comprehensive booking system that includes support for trip cancellations.
* **User Engagement:** Built-in forms for user review submissions and a feedback routing system for constant portal improvement.
* **Modern UI/UX:** Clean, responsive design featuring a dark/light mode toggle for an enhanced viewing experience.

## 🛠️ Technologies Used
* HTML5 / CSS3
* Vanilla JavaScript
* Leaflet.js (for Map Integration)

## 📁 Project Structure
* `index.html` - The main user-facing tourist portal.
* `admin.html` - The secure admin dashboard for managing destinations.
* `styles.css` - Application styling and dark mode rules.
* `app.js` - Core application logic, map rendering, and local state management.
* `/assets` - Contains project images and media.
